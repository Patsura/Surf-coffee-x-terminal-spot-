# Папка документов

Бинарные placeholder-файлы намеренно исключены из Codex PR.

Не добавляйте сюда закрытые презентации, проектную документацию арендодателя, технические чертежи, переписку по франшизе, PDF, PPTX, ZIP-архивы и другие бинарные файлы во время Codex review. Реальные документы добавляются позже вручную в GitHub, Vercel или согласованное публичное/приватное хранилище после подтверждения владельцем проекта.

Для Codex PR здесь остаются только текстовые инструкции; реальные файлы добавляются вручную после согласования доступа.
